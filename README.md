# CodeAlpha Web Scraping

## Objective
This project scrapes book information from BooksToScrape.

## Technologies Used
- Python
- BeautifulSoup
- Requests
- Pandas

## Output
The program extracts:

- Book Title
- Price
- Availability
- Rating

and stores the data in scraped_books.csv.

## Run

pip install -r requirements.txt

python web_scraping.py
# CodeAlpha - Exploratory Data Analysis (EDA)

## Objective
Perform Exploratory Data Analysis on the Titanic dataset to identify trends, patterns, and relationships.

## Dataset
Titanic-Dataset.csv

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Features
- Dataset overview
- Missing value analysis
- Data cleaning
- Descriptive statistics
- Survival analysis
- Age and fare distribution
- Passenger class analysis
- Correlation heatmap
- Visualizations saved as PNG files

## Run

```bash
pip install -r requirements.txt
python eda.py
```
